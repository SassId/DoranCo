# Basics Commands to interact with a DB with MySQL

### Version of MySQL:
```sql
mysql --version
```

### Connexion to MySQL:
```sql
mysql -u user_name -p
```
_(u is for username and p is for password)_
_(the default username is "root"; it doesn't require a password)_

### Deconnexion:
```sql
quit
```
or 
```sql
exit
```
### Display the DataBases:
```sql
SHOW DATABASES;
```

### Create a DataBase:
```sql
CREATE DATABASE db_name;
```
